Finding the Right Size Keyboard

The beauty of custom mechanical keyboards is that there are a ton of different keyboard sizes you can build. It only makes sense that several size keyboards have emerged in popularity to serve different niches. There is no universal best, just best for you.

Full-sized

Only used on pre-built keyboards, 100% keyboards have the full spaced out function keys, arrow cluster, navigation and numpad with separation to easily identify keys.

View More

1800-Compact (96%)

Only find this size through group buy, a super cool layout size that axes the navigational cluster and smushes the number pad with the rest of the keys.

View More

Tenkeyless

A common layout that chops off the number pad but still keeps the rest of the keyboard layout. The TKL still has a full separated arrow cluster, navigation and function row, so there's not much relearning that needs to happen.

View More

75%

Similar to tenkeyless, except the navigational cluster is placed vertically to save space. Fun and comfortable to use.

View More

65%

This layout removes the function row and keeps some of the navigational cluster. It's also the smallest keyboard size that keeps the dedicated arrow keys.

View More

60%

The most common custom keyboard layout. There is no function row, arrow keys, or navigational cluster.

View More

40%

The smallest size that you can still type on. It's basically a 60% keyboard except you remove the numeric keys. You'll need to get used to typing on different layers to use this size.

View More

Macropad

Basically, a separate number pad that you can assign macros to. Very fun to use alongside your main keyboard.
